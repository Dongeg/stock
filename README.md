# stock

摸鱼 股票查询 v 1.0

可以在访问百度或其他任意网站时在页面右下角查看实时股价


## 使用说明

点击右下加面板 再按 F 键弹出搜索股票输入框 

输入股票关键字回车查询，点击添加至面板

点击页面下方展开图标，显示修改网址输入框，输入网址回车修改
